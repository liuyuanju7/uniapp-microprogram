#### 软件架构
主要用到uniapp+uview1.0开发
目前只测试到了微信小程序端


#### 安装教程
1. 克隆项目
2. hbuilderx直接打开
3. 再设置中修改微信小程序的appid
![输入图片说明](/doc/appid.png)
4. 请确保后端项目已经运行[lovePig后端项目地址](https://gitee.com/jokeo/lovePigService)
5. 修改后端请求地址路径为本地
![输入图片说明](/doc//devurl.png)
6. run
